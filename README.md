# GroceryApp
Mobile app used to help make remembering grocery items even easier. No more pen &amp; paper!



## Project Requirements
The grocery app has many features!
* Data persistence
* Editing grocery items
* Deleting grocery items
* Handles null

## Tools
* Visual Studio
* Xamarin 
* Android sdk
